# UI Rules

Design:
- mobile-first
- clean
- simple
- trustworthy
- student-friendly
- card-based

Use:
- Tailwind CSS
- shadcn/ui

Core components:
- BookCard
- RequestCard
- NoteCard
- ClassroomCard
- ImageUploader
- SchoolSelector
- GradeSelector
- TrackSelector
- PriceBadge
- ConditionBadge
- FlashcardViewer
- MessageBubble

Avoid:
- excessive animations
- dark patterns
- complicated dashboards
- admin-heavy workflows

Main navigation:
- mobile-first bottom navigation
- desktop can use sidebar or top navigation
